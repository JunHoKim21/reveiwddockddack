1. 서비스 기획안 (MVP 모델)
복잡한 DOM(웹페이지 구조) 분석은 배민이나 네이버가 사이트를 업데이트할 때마다 에러를 발생시킵니다. 따라서 유지보수를 최소화하기 위해 '드래그 앤 우클릭(Context Menu)'이라는 가장 직관적이고 강력한 방식을 기획했습니다.

서비스명: 리뷰뚝딱

핵심 타겟: 배달의민족, 요기요, 네이버 스마트플레이스 리뷰를 관리하는 자영업자

핵심 동작 방식 (User Flow):

사장님이 고객의 리뷰 텍스트를 마우스로 쭉 드래그(Highlight) 합니다.

마우스 우클릭을 눌러 [리뷰요정 - AI 답글 생성] 메뉴를 클릭합니다.

화면 우측 하단에 작은 알림창이 뜨며 AI가 답글을 생성 중임을 알립니다.

생성 완료 시, 완성된 답글이 클립보드에 자동 복사되며, 사장님은 답글 입력창에 Ctrl+V만 누르면 끝납니다.

설정 화면 (Popup UI): 확장 프로그램 아이콘을 클릭하면 1) 본인의 OpenAI 또는 Gemini API Key 입력, 2) 내 가게 이름(예: "마포갈비"), 3) 답글 톤앤매너(친절하게, 유머러스하게 등)를 저장할 수 있습니다.
