# Role & Objective
너는 세계 최고의 풀스택 바이브코딩 개발자야.
자영업자용 'AI 리뷰 답글 생성기' 크롬 익스텐션(Manifest V3)과, 보안을 위한 Vercel Serverless Function(중간 API) 코드를 모두 작성해 줘.

# Architecture
- Frontend: Chrome Extension (Context Menu 방식)
- Backend: Vercel Serverless Function (Node.js)
- LLM: OpenAI API (gpt-4o-mini)

# Core Features (Backend - Vercel)
1. **`api/generate.js`**
   - Vercel에 배포될 서버리스 함수.
   - 클라이언트(크롬 익스텐션)로부터 `reviewText`, `storeName`, `tone`을 POST로 받음.
   - Vercel 환경변수(`process.env.OPENAI_API_KEY`)에 저장된 키를 사용해 OpenAI API와 통신.
   - System Prompt: "너는 [storeName]의 사장님이야. 고객 리뷰를 읽고, [tone] 느낌으로 자연스럽게 3~4문장으로 답글을 작성해. 따옴표 없이 본문만 출력해."
   - CORS 에러가 나지 않도록 적절한 Header를 추가해서 응답을 클라이언트로 반환.

# Core Features (Frontend - Chrome Extension)
1. **Popup UI (`popup.html`, `popup.js`)**
   - 사용자 API Key 입력란을 제거! 
   - 오직 '가게 이름'과 '답글 톤앤매너'만 입력받고 `chrome.storage.sync`에 저장.
2. **Context Menu (`background.js`)**
   - 텍스트를 드래그하고 우클릭하여 "[리뷰요정] AI 답글 생성"을 누르면 작동.
   - 선택된 텍스트와 저장된 '가게 이름', '톤앤매너'를 방금 만든 Vercel API 엔드포인트(예: `https://my-vercel-app.vercel.app/api/generate`)로 POST 전송.
3. **Content Script (`content.js`, `content.css`)**
   - API 통신 중에는 우측 하단에 "AI가 답글을 작성 중입니다..." Toast 알림 표시.
   - 완료되면 응답받은 텍스트를 클립보드에 자동 복사하고 Toast 알림 업데이트.

# Output Requirements
1. 전체 디렉토리 구조 (Frontend 폴더, Backend 폴더 분리).
2. Backend (Vercel) 코드: `package.json`, `api/generate.js` 코드 블록.
3. Frontend (Extension) 코드: `manifest.json`, `popup.html`, `popup.js`, `background.js`, `content.js`, `content.css` 코드 블록.
4. 코드 내 주석으로 배포 및 연동 방법에 대한 핵심 가이드를 포함해 줘.
